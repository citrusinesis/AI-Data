# AI-Data
> AI와 데이터 사회 - 강의선 교수님

### Directory Structure
```
week# (online)
├── NOTEBOOK
├── DATA
├── ...
├── lecture-note
│   ├── LECTURENOTE_PDF
│   └── ...
├── offline
│   ├── NOTEBOOK
│   ├── DATA
│   └── ...
└── worksheet
    ├── NOTEBOOK
    ├── DATA
    └── ...
```

* 모든 코드들은 Jupyter Notebook 파일(`.ipynb`)에 정리되어 있습니다.
* 로컬 Jupyter Notebook 서버는 `Python 3.10.2 64-bit`커널을 이용하였습니다.
* 코드에 사용되는 데이터는 코드와 동일한 디렉토리에 위치시켰습니다.

***이 코드를 이용하시는 분들은 지나가다 한번씩 작성자에게 감사인사를 전해시면 감사하겠습니다. 🙏❤️***
